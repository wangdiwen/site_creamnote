欢迎你加入 Creamnote 醍醐笔记网！

    醍醐笔记网 是一个专注于高校大学生的 专业课堂学习笔记、期末考试习题、考研复习资料、科创资料 的分享、交流、展示、交易于一体的网络社区平台。

    我们希望“醍醐笔记网”，可以为在校的大学生提供友好的学习服务：

    在醍醐，你可以

  •     分享、珍藏一份自己大学时代的各种优质学习笔记、考试、考研、科创方面的学习资料；
  •     通过醍醐向其他优秀的“学霸”学习，提高自己的学习成绩；
  •     通过引入审核、交易、评价、私信留言 机制，鼓励大家贡献优质的笔记内容，打造一个健康向上的资源分享平台；
  •     我们推出“Sell 笔记”，你还可以从自己的优质笔记中获得“收益”；

    我们深知大家在期末考试、考研来临时到处奔走，“各个宿舍间串门找复习资料”的“沮丧心情”，所以我们开发“醍醐笔记网”，让“学霸”们把整理的优秀学习资料、考试考研复习资料分享出来，让“屌丝”们不在为找资料而发愁啦 ~~

    同时，当学期结束、大四离校时，我们不希望看到大家把电脑里保存的珍贵资料，给“格盘”删除了，分享出来吧，说不定别人也需要呢 ~~

    当然，我们希望通过大家的共同努力，让“醍醐”成为：

  •     大学生自己的一个记录大学时代专业知识、分享、交流的网络社区；
  •     通过分享自己的“学霸”笔记，让学习成为一件有趣的事情；
  •     不论何时来到醍醐，你都会为你的学习资源帮助了学弟学妹、其他同学而感到自豪；

    在醍醐，我们还推出 特色功能 “图片笔记”，你可以将自己的手写纸质笔记资料，拍成照片上传到醍醐，创建属于你自己的电子版学习笔记。

    Helping Students Help Each Other，The Study Help You Need When You Need It.

    赶快来体验吧！
